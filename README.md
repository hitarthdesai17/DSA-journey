# 📚 DSA Journey

A structured, day-by-day log of learning **Data Structures & Algorithms in Python** — from loops and conditionals to sorting algorithms, recursion, binary search, and cyclic sort. Every topic is paired with hand-written notes and the LeetCode problems solved alongside it.

🔗 **Repo:** [DSA-journey](https://github.com/hitarthdesai17/DSA-journey)

---

## 👋 About

I'm a first-year B.Sc. AI & Data Science student. This repository is my long-term learning journal for DSA — every folder is a topic, every `readme.md` inside it is the notes I wrote while learning it, and every `.py` file is a problem I solved.

The goal isn't just to collect solutions. It's to build a reference I can come back to before interviews and actually recognize *why* a pattern applies, not just recite code.
